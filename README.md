# RS School REST service

## Clone repositorie
```
git clone git@github.com:Ortsukh/nodejs2021Q4-service.git
```
## Move to branch doker-task
```
git checkout doker-task
```
## Installing NPM modules

```
npm install
```

## Running application

```
docker-compose up
```
